# Neljandas loengus räägitud teemad

- Massiivid
  - Ühes muutujas palju elemente (muutujaid)
  - massiivi pikkus, ehk elementide arv massiivis (massiiv.length)
  - massiivi elementide indeksid algavad alati nullist
  - viimase elemendi indeks on (length - 1)
- Math-funktsioonid Javascriptis
  - Math.random()
  - Math.PI
  - Ümardamine
  - Math.floor()
  - Math.ceil()
- Funktsioonid
  - Korduvate koodilõikude kogumid
  - `function` funktsiooninimi
  - Funktsiooni keha - osa, mille sees on kood
  - Funktsiooni argumendid (sulgudes) - põhimõtteliselt muutujad funktsiooni sees
  - `return` - tagastame midagi
- Tsüklid
